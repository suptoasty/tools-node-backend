# tools-node-backend

#### Set Up
1. Clone Repo
2. `npm install`
3. `npm run dev`

#### CRUD Commands / Routes
GET Request: `localhost:3001/coursesapi/courses/:id`  
GET Result:  `{
    "course_id": #,
    "course_dept": "",
    "course_num": "",
    "course_level": "",
    "course_hours": "",
    "course_name": "",
    "course_desc": ""
}`
